Анализ сайта выполняется через веб-форму POST / с полем url. Free-план: {{ free_daily_limit }} анализов в сутки на IP. Полные метрики /api/stats — только localhost или заголовок X-Admin-Token.

Системные

GET /health

Health-check для reverse proxy и мониторинга.

{"status": "ok"}
GET /version

Текущая версия приложения.

{"version": "{{ app_version }}"}

История и аналитика

GET /api/history?limit=10

Список последних анализов с дельтами изменений.

Параметры: limit (int, default 10)

GET /api/history/filter

Фильтрация истории по дате, SEO, домену, избранному.

Параметры: date_from, date_to, seo_score, load_time_min, load_time_max, domain, favorites_only, tags

GET /api/stats

Публичная статистика. Расширенные метрики — с admin-токеном.

curl -H "X-Admin-Token: YOUR_TOKEN" {{ request.url_root }}api/stats
GET /api/analytics

Данные для графиков на странице истории.

Управление

DELETE /api/delete-analysis/<id>

Удалить одну запись из истории.

DELETE /api/clear-history

Очистить всю историю анализов.

POST /api/favorites

Переключить избранное для записи.

{"analysis_id": 1, "is_favorite": true}
GET /api/alerts

Список алертов (ухудшение SEO, производительности).

POST /api/alerts

Отметить алерт прочитанным.

{"alert_id": 1}
